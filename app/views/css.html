To animate without the need to hover...

.hvr-grow:hover, .hvr-grow:focus, .hvr-grow:active { -webkit-transform: scale(1.1); transform: scale(1.1); }

becomes just the class name..

.hvr-grow { -webkit-transform: scale(1.1); transform: scale(1.1); }

However! if the animation is not a looping animation ("Pulse" loops where "Grow" does not.) then you cannot do the above, else the animation will skip to the end of the animation, looking turd.
